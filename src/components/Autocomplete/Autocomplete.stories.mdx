# Autocomplete

This component allows us to suggest inputs to the user. It displays a list of strings, if none are provided it behaves like a normal text field. It is based on [Material UI Autocomplete](https://mui.com/material-ui/react-autocomplete/).

export const Template = (args) => <Autocomplete {...args}>Button</Autocomplete>;
