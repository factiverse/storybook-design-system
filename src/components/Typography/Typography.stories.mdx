# Typography

Wrapper for [Material UI Typography](https://mui.com/material-ui/react-typography/).
The same properties can be passed to this component (see the Args Table below) but we prefer to use the following composition in most cases.

## Primary Typeface: Arizona Serif

Arizona Serif is the primary typeface and is used for headlines and body text to increase readability in small sizes.

## Secondary Typeface: DM Mono

DM Mono is used on smaller elements like overline headings and buttons, and should not be used in big sizes.
