# Buttons

We use wrapped Material UI Buttons. The same properties can be passed to this component (see the Args Table below) but we prefer to use the following default variations in most cases.

## Usage

- `color="primary"` for the most common interactions
- `color="secondary"` for important interactions to **draw attention**

## Interaction Test

export const Template = (args) => <Button {...args}>Button</Button>;

<Canvas>
  <Story
    name="Button click"
    play={async ({ args, canvasElement }) => {
      const canvas = within(canvasElement);
      await userEvent.click(canvas.getByRole('button'));
      await waitFor(() => expect(args.onClick).toHaveBeenCalled());
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
