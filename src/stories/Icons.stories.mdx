## Usage

- Always use the "light" version
- Download SVGs instead of raster images
- Add them in React/HMTL by adding a class with the name of the icon:
  ```
    <i className="ph-address-book-light"></i>
  ```
- Use [helper classes](https://github.com/phosphor-icons/phosphor-icons) to change the size of the icon:
  ```
    <i className="ph-address-book-light ph-4x"></i>
  ```
